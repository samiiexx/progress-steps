# Progress Steps
A JavaScript Project from Brad Traversy's 50 Projects in 50 Days. This project is focused on building progress steps. Clicking the 'next' or 'prev' button counts to the next or previous number. 

![Progress Steps](screenshot.jpg)

## Built With
- [HTML5](https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/HTML5)
- [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS)
- [Javascript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

## Install
```
    $ git clone git@github.com:samiiexx/sam-js-2.git
    $ cd main
```
